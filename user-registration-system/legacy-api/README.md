# legacy-api

## scaffolding

```shell
mkdir legacy-api
cd legacy-api
npm init --yes
nano package.json
touch .gitignore
nano .gitignore
touch HELP.md
touch README.md
nano README.md
touch .env
nano .env
```

## proceed to install some dependencies

```shell
npm view @contentpi/lib versions
npm i @contentpi/lib@1.0.2
npm view @graphql-tools/load-files versions
npm i @graphql-tools/load-files@6.3.2
```
